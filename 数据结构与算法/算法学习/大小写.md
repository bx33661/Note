> 要将字符串中的所有字母转换为小写，你可以使用字符串的 **lower()** 方法。下面是一个示例：

```python
string = "Hello World"
lowercase_string = string.lower()
print(lowercase_string)
```
输出将是：
```python
hello world
```
在这个例子中，**lower()** 方法被调用并应用于字符串 **string**，它将字符串中的所有字母转换为小写形式并返回一个新的字符串
