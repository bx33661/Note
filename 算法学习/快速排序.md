# 快速排序

```python
def quick_sort(arr):
    if(len(arr)<=1):
        return arr
    else:
        n=arr[0]
        less_arr=[x for x in arr[1:] if x<=n]
        bigger_arr=[x for x in arr[1:] if x>n]
        return quick_sort(less_arr) + [n] +quick_sort(bigger_arr)

li=[55,22,88,66,77,33]
print(quick_sort(li))
```
